# rm-rf.bat

Утилита командной строки для Windows, имитирующая поведение команды `rm -rf` из Unix-систем. Позволяет рекурсивно удалять файлы и каталоги без подтверждения.

## Особенности

- Поддержка удаления нескольких файлов и папок за один вызов
- Автоматическое определение типа объекта (файл или каталог)
- Отсутствие запросов на подтверждение (аналог `-f` и `-r`)
- Простое использование из командной строки
- Поддержка wildcard-масок (`*` и `?`) для удаления групп файлов

## Требования

- Windows (любая версия с поддержкой `cmd.exe`)
- Файл `rm-rf.bat` должен быть сохранён в кодировке **CP866**, чтобы корректно отображался русский текст в командной строке

## Установка

1. Скопируйте файл `rm-rf.bat` в любую папку, входящую в переменную окружения `PATH`, например:
   - `C:\Windows\System32`
   - `C:\Tools`
   - Добавьте отдельную папку в `PATH` и поместите туда скрипт

2. Перезапустите командную строку, чтобы изменения вступили в силу

## Использование

```cmd
rm-rf "C:\путь\к\папке"
```

или

```cmd
rm-rf "file.txt" "folder" "C:\temp\old"
```

или с использованием масок:

```cmd
rm-rf "*.tmp"
rm-rf "backup_??.zip"
```

## Аргументы

Скрипт принимает один или несколько путей:

- Если путь указывает на каталог — он удаляется рекурсивно с помощью `rmdir /s /q`
- Если путь указывает на файл — он удаляется с помощью `del /f /q`
- Если путь не существует — выводится предупреждение, но выполнение продолжается
- Поддержка wildcard-выражений (`*` и `?`) для удаления нескольких файлов по шаблону

## Примеры

Удаление папки:
```cmd
rm-rf "C:\Temp\OldProject"
```

Удаление нескольких файлов и папок:
```cmd
rm-rf "file1.log" "cache" "C:\temp\backup"
```

Удаление временных файлов:
```cmd
rm-rf "*.tmp"
```

Удаление архивов с шаблоном имени:
```cmd
rm-rf "backup_??.zip"
```

## Кодировка и кириллица

> **Важно:** Файл `.bat` должен быть сохранён в кодировке **CP866**, иначе сообщения на русском языке будут отображаться как кракозябры в стандартной командной строке Windows (`cmd.exe`).

Если вы используете PowerShell, можно сохранить в UTF-8 с BOM, но в `cmd.exe` это может вызвать проблемы.

## Возврат кодов ошибок

- `0` — всё успешно удалено
- `1` — ошибка удаления или не указаны аргументы

## Внимание

Этот скрипт **не отправляет файлы в корзину** — он удаляет их **навсегда**. Используйте с осторожностью!
